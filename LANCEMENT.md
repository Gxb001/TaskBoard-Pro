# Lancement du projet

## Prerequis

- Node.js (v18+)

## Commandes

| Systeme              | Commande      |
|----------------------|---------------|
| Windows (PowerShell) | `.\start.ps1` |
| Windows (CMD)        | `start.bat`   |
| Linux/macOS          | `./start.sh`  |

## Acces

Ouvrir http://localhost:4200 dans le navigateur.
